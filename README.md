# Financial Management Presentation

This is a GitHub Pages-friendly interactive presentation for new trainees on saving, investing, financial products, and portfolio thinking.

## Run locally

Open `index.html` in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Presenting

- Use arrow keys to navigate.
- The URL hash updates slide-by-slide, similar to other Reveal-style decks.
- The deck includes live sliders for budgeting, SIP compounding, and portfolio construction.

## Deploy on GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, go to `Settings -> Pages`.
3. Choose `Deploy from a branch`.
4. Select your main branch and the root folder.
5. Save, then open the generated Pages URL.

## Notes

- The content is India-oriented because it covers PPF, VPF, demat, SGB, SEBI, NSE, and BSE.
- This is educational material, not personal financial advice.
- Rates, tax rules, and issuance windows change over time, so verify the latest official notices before presenting time-sensitive numbers as current.
